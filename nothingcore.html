Nothing Core

Nothing Core is a core mod designed for the modpacks by Nothing Series.
It adds missing items, features, and datapacks to enhance world generation and improve compatibility between mods.
These additions create a smoother, more cohesive modpack experience, allowing various content mods to work together more harmoniously.

W.I.P.

The Mod is still work in progress and currently in alpha. You can download the alpha for 1.20.1 on the Discord.
